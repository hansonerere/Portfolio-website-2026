# 🧪 内容更新测试指南

## 🎯 快速测试内容同步

### 测试1：修改首页服务卡片
1. 登录 Supabase Dashboard
2. 进入 Table Editor → `service_cards`
3. 找到第一个服务卡片
4. 修改 `title` 字段，比如改为 "Brand Design (Updated)"
5. 点击保存
6. 刷新您的网站 → 应该能立即看到更改！

### 测试2：更新About页面内容
1. 进入 `about_cards` 表
2. 选择任意一个卡片
3. 修改 `content` 字段
4. 保存后刷新网站检查

### 测试3：添加新项目
1. 进入 `projects` 表
2. 点击 "Insert" → "Insert row"
3. 填写新项目信息
4. 保存后在网站左侧导航应该能看到新项目

## ⚡ 同步时间测试

**预期结果**：
- 修改后 **5-15秒** 内在网站上看到更新
- 无需重新部署
- 全球用户都能看到最新内容

## 📱 最佳实践

### ✅ 推荐操作：
- 在Supabase中批量修改多个字段
- 使用有效的图片URL（确保链接可访问）

### ⚠️ 注意事项：

- 图片URL必须可公开访问
- JSON格式的字段要保持正确格式
